# CHANGELOG

## v9.0.0 (2026-04-30)

### ✨ 主要更新

- **#更改人设 命令**: 普通用户可以直接在 QQ 群中修改自己的人设
- **人设即时生效**: 修改后立即生效，无需重启机器人
- **人设长度无限制**: 支持任意长度的人设内容
- **状态验证**: 必须已登录并运行中才能修改人设

### 🔧 技术改进

- 新增 `savePersona()` 函数保存用户人设
- 新增 `changePersona()` 命令处理方法
- 更新命令匹配规则，支持 `#更改人设 人设内容` 格式
- 添加账号和运行状态验证逻辑
- 更新帮助信息，包含新命令说明
- 更新所有文档，详细说明新功能使用方法

## v1.0.0 (2026-04-30)

### ✨ 新增功能

- **多用户独立登录系统**: 每个用户独立登录微信，独立环境
- **独立配置管理**: 每个用户有自己的persona、记忆、配置文件
- **QQ群扫码登录**: 在QQ群中发送 `#登录微信AI` 获取二维码
- **主人权限管理**: 配置master.json管理所有用户
- **在线列表查看**: `#微信机器人在线列表` 查看所有运行中的机器人
- **实例启停控制**: `#启动机器人` / `#停止机器人` 管理实例
- **账号删除功能**: `#删除机器人` 清理用户数据

### 📁 目录结构

```
AI-MultiUser-Bot/
├── plugin/
│   ├── 主插件.js            # 主命令处理
│   └── 微信消息处理.js      # AI对话处理
├── accounts/                # 用户数据目录
│   └── user-XXX/
│       ├── config.json
│       ├── persona.md
│       └── chat-memory.json
├── default-persona.md       # 默认人设模板
├── masters.json            # 主人配置
├── README.md
└── CHANGELOG.md
```

### 🎯 核心特性

- 每个用户独立的聊天记忆（保留3天）
- 每个用户独立的人设设置
- 消息分段发送，模拟真实聊天
- 自动清理过期聊天记录
- 基于现有openclaw-weixin架构

### 📝 命令列表

**普通用户命令**:
- `#登录微信AI` - 获取二维码登录

**主人命令**:
- `#微信机器人在线列表` - 查看所有在线机器人
- `#停止机器人 [用户ID]` - 停止指定机器人
- `#启动机器人 [用户ID]` - 启动指定机器人
- `#删除机器人 [用户ID]` - 删除账号数据
- `#帮助多用户` - 查看帮助
