## 快速开始

### 前置条件

1. ✅ 已安装 XRK-AGT 框架
2. ✅ 已配置 NapCat 或其他 OneBot 客户端
3. ✅ 已在 `plugin-config.json` 中配置 API Key
4. ✅ Puppeteer 浏览器已正确配置（用于截图二维码）

### 安装步骤

1. 将 `AI-MultiUser-Core` 文件夹放置到 `XRK-AGT/core/` 目录下
2. 编辑 `plugin-config.json` 配置 API Key 和模型
3. 编辑 `masters.json` 配置主人 QQ 号
4. 启动 XRK-AGT 机器人

## 特色功能

### 🎯 真人聊天体验

#### 多段发送
- AI回复自动分成短段落（最多10段）
- 按句子和段落智能拆分
- 更符合微信聊天习惯

#### 模拟打字
- 0.5-1秒的随机延迟
- 每段独立延迟发送
- 模拟真人打字节奏

#### 真人身份
- 牢记自己是一个人，不是AI或机器造物
- 无论谈论什么话题都保持真人身份
- 避免任何AI或机器相关的表达

#### 例句参考
- 发言例句仅供参考
- 无需按照示例格式输出
- 保持自己的说话风格

#### 说话限制
- 日常对话：1句话，或总字数不超过30字
- 专业/探讨内容：最多3句话，或总字数不超过50字
- 严格控制说话长度

#### 句末句号
- 说话请去掉句末的句号
- 保持自然的口语表达
- 更符合真人聊天习惯

#### 珍惜机会
- AI会精简回复，只说最重要的内容
- AI珍惜每一个回复机会
- 最多发送不超过10段

#### 人设优先
- 严格按照用户人设说话
- 人设是最高优先级
- 人设冲突时以人设为准

#### 回复丰富
- 在限制下充分表达
- 回复内容充实，不要太简短
- 平衡分段与内容丰富度

#### 自然对话
- AI提示词引导分段回复
- 每段控制在10字以内
- 段落之间用空行分隔

#### 日志监控
- 详细的发送进度日志
- 显示每段发送状态
- 显示延迟时间

## 命令说明

### 普通用户命令

| 命令 | 说明 | 使用场景 |
|------|------|----------|
| `#登录微信AI` | 开始微信登录流程，获取二维码图片 | 新用户首次登录 |

### 主人命令

| 命令 | 说明 | 参数 |
|------|------|------|
| `#微信机器人在线列表` | 查看所有已登录用户和运行状态 | 无 |
| `#停止机器人` | 停止自己的机器人 | 无 |
| `#停止机器人 [QQ号]` | 停止指定用户的机器人 | QQ号（可选） |
| `#启动机器人` | 启动自己的机器人 | 无 |
| `#启动机器人 [QQ号]` | 启动指定用户的机器人 | QQ号（可选） |
| `#删除机器人` | 删除自己的所有数据（删除文件夹） | 无 |
| `#删除机器人 [QQ号]` | 删除指定用户的所有数据（删除文件夹） | QQ号（可选） |
| `#帮助多用户` | 查看帮助信息 | 无 |

## 使用流程

### 普通用户使用流程

```
1. 在QQ群中发送 #登录微信AI
   ↓
2. 收到二维码图片
   ↓
3. 用微信扫码登录
   ↓
4. 登录成功后自动启动机器人
   ↓
5. 在微信中和机器人对话（体验真人聊天节奏）
```

### 主人管理流程

```
1. 编辑 masters.json 配置主人QQ号
   ↓
2. 在QQ群中使用管理命令
   ↓
3. 查看和控制所有机器人
```

### 自定义人设流程

```
1. 编辑 default-persona.md（修改新用户默认人设）
   ↓
2. 或编辑 accounts/user-XXX/persona.md（修改个人人设）
   ↓
3. 立即生效，无需重启
```

## 常见问题

### Q: 为什么登录后机器人不回复？

A: 请检查以下几点：
1. 查看控制台日志是否有错误
2. 确认微信已正确扫码登录
3. 检查 `plugin-config.json` 中的 API Key 是否正确

### Q: 二维码发送失败怎么办？

A: 请检查 Puppeteer 浏览器是否正确配置，如果图片发送失败，会降级发送链接。

### Q: 聊天记忆保存多久？

A: 默认保存 7 天，过期自动清理。

### Q: 如何自定义人设？

A: 编辑 `accounts/user-XXX/persona.md` 文件即可，立即生效。

### Q: 主人如何管理用户？

A: 在 `masters.json` 中配置主人 QQ 号后，可在 QQ 群中使用管理命令。

### Q: 删除机器人会删除什么？

A: 会删除用户的整个数据文件夹，包括配置、人设、聊天记忆等所有数据。

### Q: 如何重启机器人？

A: 使用 `#停止机器人` 后，再用 `#启动机器人` 即可。

### Q: 支持多个微信账号同时登录吗？

A: 支持！每个 QQ 用户可以登录一个独立的微信账号，互不干扰。

### Q: AI回复为什么分成好几段？

A: 这是为了模拟真人聊天体验，每段之间有0.5-0.8秒的延迟，更像真人打字的节奏。

### Q: 可以调整发送延迟时间吗？

A: 目前延迟固定在0.5-0.8秒随机范围，后续版本可能会支持自定义配置。

### Q: AI回复太短了，可以让回复长一点吗？

A: 可以！修改人设文件，让AI倾向于生成更长的回复，但仍会保持分段发送的特性。

## 安全提示

⚠️ 不要将 `plugin-config.json` 中的 API Key 泄露给他人
⚠️ 不要在公共平台分享包含 token 的配置文件
⚠️ 及时更新 API Key 保证安全
⚠️ 不要在 QQ 群中发送敏感信息
