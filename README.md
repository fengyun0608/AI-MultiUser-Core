## 📂 每个目录/文件的详细作用

### 1. `plugin/` - 插件目录
| 文件名 | 作用 |
|--------|------|
| `多用户微信机器人.js` | 核心插件，处理QQ群内的所有命令、登录流程、账号管理、微信消息监听、AI对话、人设管理、聊天记忆、消息回复 |

### 2. `accounts/` - 用户数据根目录（完全隔离）
每个用户都有自己的独立文件夹，数据互不干扰！

### 3. `user-XXX/` - 单个用户的数据目录（XXX为QQ号）
| 文件名 | 作用 | 隔离性 |
|--------|------|--------|
| `config.json` | 存储用户的微信登录信息、token、状态配置 | ✅ 完全独立 |
| `persona.md` | 用户专属的人设文件，可自定义角色性格 | ✅ 完全独立 |
| `chat-memory.json` | 用户的聊天记忆，保留最近7天对话 | ✅ 完全独立 |

### 4. `docs/` - 文档目录
| 文件名 | 作用 |
|--------|------|
| `CONFIG.md` | 配置说明文档，包含API配置、人设配置等 |
| `USAGE.md` | 使用说明文档，详细的使用指南 |
| `API.md` | API说明文档，接口说明 |

### 5. 根目录文件
| 文件名 | 作用 | 可修改 |
|--------|------|--------|
| `default-persona.md` | 默认人设模板，新用户登录时自动复制 | ✅ 可自定义 |
| `plugin-config.json` | 插件配置文件（API配置、LLM配置） | ⚠️ 仅管理员 |
| `masters.json` | 主人QQ号列表，配置管理员权限 | ⚠️ 仅管理员 |
| `README.md` | 说明文档 | |
| `CHANGELOG.md` | 版本更新记录 | |

## 🔒 隔离性保证

✅ **每个人设完全隔离** - 每个用户有自己的 `persona.md`
✅ **每个配置完全隔离** - 每个用户有自己的 `config.json`
✅ **每个记忆完全隔离** - 每个用户有自己的 `chat-memory.json`
✅ **数据互不干扰** - 用户之间无法访问对方的数据
✅ **安全日志输出** - 日志中不暴露token、user_id等敏感信息

## ✨ 特色功能

### 🎯 真人聊天体验
- **多段发送**: AI回复自动分成短段落（最多10段）
- **模拟打字**: 0.5-1秒的随机延迟
- **珍惜机会**: AI会精简回复，只说最重要的内容
- **人设优先**: 严格按照用户人设说话，人设是最高优先级
- **回复丰富**: 在10段限制下充分表达，回复内容充实
- **自然节奏**: 更像真人打字的感觉
- **智能分段**: 按句子和段落自动拆分

## ⚙️ 配置说明

### 插件配置 - `plugin-config.json`（仅管理员修改）

```json
{
  "apiUrl": "https://api.minewuer.com/v1/chat/completions",
  "apiKey": "sk-xxx",
  "model": "deepseek-v4-pro-chat",
  "temperature": 0.7,
  "maxTokens": 1000
}
```

### 主人配置 - `masters.json`

```json
[
  "123456789",
  "987654321"
]
```

将主人的QQ号添加到此列表，即可使用管理命令。

---

### 用户配置 - `config.json`（自动生成，无需手动编辑）

```json
{
  "userId": "123456789",
  "accountId": "wxid_xxx@im.bot",
  "token": "xxx",
  "baseUrl": "https://ilinkai.weixin.qq.com",
  "userIdFromWeixin": "xxx",
  "createdAt": 1234567890,
  "enabled": true,
  "get_updates_buf": ""
}
```

---

### 人设文件 - `persona.md`（可自定义）

每个用户登录后，会自动复制 `default-persona.md` 作为初始人设。你可以：
1. 直接编辑 `default-persona.md` 来修改新用户的默认人设
2. 登录后编辑 `accounts/user-XXX/persona.md` 来自定义个人人设

## 🚀 使用流程

### 普通用户使用
1. 在QQ群中发送 `#登录微信AI`
2. 收到二维码图片
3. 用微信扫码登录
4. 登录成功后自动启动机器人
5. 在微信中和机器人对话（体验真人聊天节奏）

### 主人管理
1. 编辑 `masters.json` 配置主人QQ号
2. 在QQ群中使用管理命令查看和控制所有机器人

## 🛠️ 系统架构说明

### 插件协作方式
- **多用户微信机器人.js**：负责QQ群命令、微信登录、消息监听、AI对话、人设管理、消息回复
- 单插件架构，无需其他插件依赖

### 消息流程
1. 用户在微信发送消息
2. 多用户微信机器人.js监听并接收消息
3. 调用AI生成回复
4. 智能分段成短段落
5. 每段0.5-0.8秒延迟发送
6. 调用微信API发送回复
